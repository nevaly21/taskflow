# TaskFlow 3.0

TaskFlow 3.0 es una plataforma académica diseñada para demostrar la implementación de arquitecturas cloud escalables y seguras utilizando servicios de Amazon Web Services (AWS) bajo el modelo IaaS (Infraestructura como Servicio).

## Objetivo del Proyecto

Desarrollar una solución que integre conceptos del modelo NIST de computación en la nube y servicios clave de AWS como EC2, S3, IAM, VPC, y Lambda, para construir una infraestructura automatizada, medible y adaptativa.

## Servicios Usados

- **Amazon EC2**: Para instancias virtuales de cómputo.
- **Amazon S3**: Almacenamiento duradero y escalable de archivos.
- **AWS IAM**: Control de accesos y roles.
- **AWS Lambda**: Ejecución de funciones automáticas sin servidores.
- **Amazon VPC**: Red privada virtual para mayor seguridad.
