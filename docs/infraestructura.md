# Infraestructura del Proyecto TaskFlow 3.0

Este proyecto está diseñado sobre una arquitectura IaaS utilizando servicios de Amazon Web Services (AWS).

## Componentes utilizados:

- **EC2**: para instancias de computación.
- **S3**: para almacenamiento seguro de archivos.
- **IAM**: para la gestión de identidades y control de acceso.
- **VPC**: para configurar redes privadas.
- **Lambda**: para automatización de tareas sin servidores.
- **Auto Scaling y ELB**: para escalabilidad y balanceo de carga.

Se sigue el modelo NIST: acceso bajo demanda, elasticidad y servicio medido.
