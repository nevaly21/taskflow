# Justificación de Herramienta

Se eligió AWS por ser una de las plataformas cloud más completas y accesibles para ambientes académicos. Sus servicios permiten construir arquitecturas seguras, escalables y de bajo costo inicial.

Entre las razones destacan:
- Control granular de usuarios con IAM.
- Elasticidad automática con Auto Scaling.
- Amplio ecosistema y documentación.
- Uso gratuito de muchos servicios en planes estudiantiles.
